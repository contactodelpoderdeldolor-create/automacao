📌 Como funciona

1. Sistema recebe webhook de PIX gerado

2. Registra os dados do cliente

3. Aguarda o tempo configurado

4. Verifica se o pagamento ainda está pendente

5. Se pendente, envia mensagem via WhatsApp

⚠️ Se o cliente responder, a automação é pausada automaticamente.
